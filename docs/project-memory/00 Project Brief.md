# Tradex Project Brief

## Purpose

Tradex is a kids-friendly multiplayer marketplace simulation where players share one persistent economy, each own one shop, buy stock from a supplier, publish listings, compete on price and rating, and sell to both human players and bots.

## Current Product Reality

This is a continuation project, not a fresh build. The app already has working core marketplace flows, live production hosting, shared persistent data, and existing repo documentation.

## Current Stack

- Frontend and backend: Next.js App Router
- Language: TypeScript
- ORM: Prisma
- Database engine: PostgreSQL
- Hosted database provider: Neon
- Hosting platform: Vercel
- Preferred production region: `syd1`

## Working Product Scope

- Account registration and password login
- Create-shop onboarding
- Supplier purchasing into inventory
- Listing creation and management
- Marketplace browsing, search, sorting, and filters
- Shop pages
- Single-seller cart and checkout
- Persistent orders, notifications, balances, and inventory changes
- Bot purchases, demand shifts, supplier repricing, and market events
- Production deployment

## Current Focus

- Improve performance and responsiveness
- Keep production data clean and secure
- Improve continuity and documentation
- Streamline future work without rebuilding working systems
